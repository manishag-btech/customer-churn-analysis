# Customer Churn Analysis

## Project Overview
This project analyzes customer churn data to identify patterns and factors affecting customer retention.

## Dataset
- Dataset: Telco Customer Churn Dataset
- Total Records: 7043

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- VS Code

## Analysis Performed
- Data Cleaning
- Missing Value Analysis
- Duplicate Value Check
- Exploratory Data Analysis (EDA)

## Visualizations
1. Customer Churn Distribution
2. Gender vs Churn
3. Contract Type vs Churn
4. Monthly Charges vs Churn

## Key Insights
- Most customers did not churn.
- Customers with month-to-month contracts are more likely to churn.
- Higher monthly charges are associated with higher churn.
- Gender has minimal impact on churn.

## Conclusion
Customer churn is strongly influenced by contract type and monthly charges. Businesses can improve customer retention by focusing on customers with month-to-month contracts and high monthly charges.
